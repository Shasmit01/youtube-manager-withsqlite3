# YouTube Manager App (Python + SQLite3)

A simple command-line based YouTube video manager built using Python and SQLite3.  
This project demonstrates CRUD operations (Create, Read, Update, Delete) using a relational database.

## Features
- Add video details (name and duration)
- View all saved videos
- Update video information
- Delete videos
- Data stored using SQLite3 database
- Menu-driven CLI application

## Tech Stack
- Python
- SQLite3
- VS Code

## Learning Outcomes
- Understanding of database integration in Python
- Performing CRUD operations using SQL
- Writing modular Python functions
- Building real-world beginner projects
- Debugging syntax and logical errors
